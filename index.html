Arithmetic to Assembly

This was a project to learn the basics of the compiler pipline, and to examine the difficulties with generating arbitrary ARM assembly.

This is interactive. Play around with the sample input, and check the generated ASM.


Features

  • Addition / Subtraction
  • Multiplication / Integer Division
  • Exponents
  • Nested Expressions

Technical Details

  • Written in Go
  • Github Repo
  • Compiled to Javascript using GopherJS
  • Made by Harrison Turton
INPUT 
main:
  mov r0, 1
  push {r0}
  mov r0, 2
  push {r0}
  pop {r0, r1}
  add r0, r1
  push {r0}
  mov r0, 4
  push {r0}
  pop {r1, r2}
  mov r0, 1
  bl exponent
  push {r0}
  b finish
exponent:
  cmp r1, 0
  beq branchToLR
  mul r0, r2
  sub r1, 1
  b exponent
branchToLR:
  bx lr
finish:
  nop
  b finish
